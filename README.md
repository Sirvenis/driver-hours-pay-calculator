# Driver Hours & Pay Calculator

An installable mobile Progressive Web App for company drivers to track shifts and calculate gross pay.

Features:

- Start time and finish time
- Unpaid break minutes
- Hourly pay rate
- Overnight shift support
- Saved shift entries on the phone
- Weekly, fortnightly, and monthly pay period totals
- Offline support after first load

## Install on Phone

Open the GitHub Pages URL on your phone.

### iPhone
1. Open in Safari
2. Tap Share
3. Tap Add to Home Screen
4. Tap Add

### Android
1. Open in Chrome
2. Tap the menu
3. Tap Install app or Add to Home screen
